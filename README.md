
# 💬 Slack Clone

A full-stack Slack clone built as a portfolio project to demonstrate real-time communication, user authentication, and channel-based messaging. This project mimics core features of Slack and is built using **Java (backend)**, **React.js (frontend)**, and **PostgreSQL (database)**.

## 🚀 Features

- ✅ User registration and authentication
- 🧑‍🤝‍🧑 Create and join channels
- 💬 Real-time messaging
- 📨 Direct messages between users
- 📌 Message history and persistence
- 🌐 RESTful APIs with secure endpoints

## 🛠️ Tech Stack

| Layer       | Technology       |
|-------------|------------------|
| Frontend    | React.js         |
| Backend     | Java (Spring Boot) |
| Database    | PostgreSQL       |
| Realtime    | WebSocket (SockJS/STOMP) |
| Auth        | JWT (JSON Web Tokens) |
